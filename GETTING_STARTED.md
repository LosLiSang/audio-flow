## 🚀 3 分钟快速开始

### 第一步：安装依赖

```bash
# 安装 Tauri CLI（仅需一次）
cargo install tauri-cli

# 安装 npm 依赖
npm install
```

### 第二步：启动应用

```bash
npm run tauri dev
```

首次运行需要 5-10 分钟编译。

### 第三步：开始使用

1. 点击左侧"刷新设备"查看音频设备
2. 点击"添加路由"创建音频路由
3. 点击"启动"开始混音
4. 查看右下角 VU 表监控音频

## 💡 使用提示

### VB-Cable 设置

1. **安装 VB-Cable**
   - 访问 https://vb-audio.com/Cable/
   - 下载并安装 VB-Cable
   - 重启系统

2. **配置 Audio Flow**
   - 在 Audio Flow 中添加路由
   - 输入选择您的麦克风/音频源
   - 输出选择 VB-Cable

3. **使用音频**
   - 在其他应用（如 OBS）中选择 VB-Cable 作为输入
   - 音频将从您的源设备传输到应用

### 性能优化

- **降低延迟**：编辑 `src-tauri/src/audio/engine.rs`，将缓冲区从 512 改为 256
- **提高稳定性**：将缓冲区改为 1024
- **调整增益**：避免超过 0dB 以防失真

## 🐛 常见问题

### Q: 编译失败怎么办？

```bash
# 清理并重新编译
cd src-tauri
cargo clean
cargo check
```

### Q: 找不到音频设备？

1. 检查 Windows 音频设置
2. 确保设备已启用
3. 点击"刷新设备"按钮

### Q: 延迟太高怎么办？

1. 调整缓冲区大小（见上方性能优化）
2. 关闭其他占用 CPU 的程序
3. 使用独占模式（高级设置）

### Q: VB-Cable 不工作？

1. 确保 VB-Cable 已正确安装
2. 重启 Windows 音频服务
3. 检查是否被其他应用占用
